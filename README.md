# Project_Showcase_App
A React SPA for managing an e-commerce product catalogue. Built with Vite, React Router v6, and json-server as a simulated backend.

## Features
- Landing page with site overview
- Product list with live search
- Product detail page with inline editing
- Add new product form
- Delete products
- Client-side routing

## Tech Stack
- React 19 + Vite
- React Router v6
- json-server 

## Getting Started

### 1. Install dependencies
```bash
npm install
```

### 2. Start the mock backend (port 3001)
```bash
npm run server
```

### 3. Start the React dev server (port 5173)
```bash
npm run dev
```

### 4. Run tests
```bash
npm test
```
